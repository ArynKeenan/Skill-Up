# Skill Up v1.0.0
### Bridging the gap between your talent and your next role.

## 🚀 Overview
**Skill Up** is a Chrome Extension designed to take the guesswork out of job hunting. By analyzing job descriptions in real-time, it identifies exactly which technical skills you already possess and which ones you need to master to land the interview.

## 🛠️ How It Works
Skill Up bridges the gap between your talent and your next role by analyzing job descriptions in real-time. Click **Compare** to instantly scan any listing against your saved profile for a compatibility score. Use the **Add Skills** modal to input your tech stack, while the **Remove All** feature utilizes a safety-first "double-click" mechanic to prevent accidental data loss.

**⚠️ Note on Accuracy:** This tool provides a high-level overview based on our current skill library. Niche technologies or non-standard terms may not be captured. Use this score as a helpful guide for your initial review, but always perform a final manual check of the job requirements before applying.

## ✨ Key Features
* **Instant Comparison:** Real-time matching logic against active job boards.
* **Custom Skill Vault:** Add and manage your personal technical stack.
* **Data Safety:** State-based confirmation for clearing your profile.
* **Privacy-First:** All data is stored locally in your browser (`chrome.storage.local`).

## 💻 Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **API:** Chrome Extension Manifest V3
* **Storage:** Chrome Storage API

## 📥 Installation (Developer Mode)
1. Clone this repository to your local machine.
2. Open Chrome and go to `chrome://extensions`.
3. Toggle **Developer mode** (top right).
4. Click **Load unpacked** and select the project folder.
